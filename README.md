# 🌾 Agriculture Shop Management System

A desktop application developed with Python and Tkinter for managing an agricultural tools store.

## ✨ Features

- 👤 Customer information
- 🌾 Product management
- ➕ Add products
- ✏️ Edit products
- 🗑 Delete products
- 🔍 Search products
- 📊 Invoice calculations
- 💰 Automatic tax calculation
- 🖨 Print invoice
- 🕒 Live clock
- 📋 Status Bar
- 📑 New Invoice

## 🛠 Technologies

- Python
- Tkinter
- ttk
- Treeview
